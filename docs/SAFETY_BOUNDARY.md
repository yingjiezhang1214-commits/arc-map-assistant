# Safety Boundary

This project is an independent desktop map assistant. It is designed to stay outside the game process and use only ordinary desktop techniques.

## This Tool Uses

- Ordinary screen-space overlay windows.
- Local JSON configuration.
- User-triggered hotkeys.
- Ordinary Windows screenshots manually triggered by the user.
- In later phases, OpenCV image recognition on screenshots.

## This Tool Does Not Use

- Game memory reading.
- Process injection.
- DLL injection.
- DirectX, Vulkan, or OpenGL hooks.
- Driver-level capture.
- Keyboard or mouse simulation.
- Automatic gameplay actions.
- Enemy, player, health, or bullet-line recognition.
- Network packet analysis.
- Anti-cheat bypasses.
- Process hiding.

## Risk Statement

Third-party overlays may still be considered risky by game publishers or anti-cheat systems. This project does not promise complete zero-ban safety. The safety boundary reduces risk by avoiding invasive behavior; it cannot guarantee absolute safety.
